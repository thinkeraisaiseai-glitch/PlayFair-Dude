# Contributing to PlayFair Dude

Thank you for your interest in contributing to PlayFair Dude.

This project is **source-available and founder-governed**. Contributions are
welcome, but must follow strict ethical and legal boundaries to protect players,
developers, and the project itself.

We accept contributions that:
- Improve documentation or clarity
- Strengthen privacy protections

- Enhance transparency and auditability
- Identify design flaws or ethical risks
- Improve performance or stability

All contributions must align with:
- Player-first design
- Non-invasive data collection
- Defensive security only

We do NOT accept contributions that:

- Circumvent platform rules or protections

Submissions violating these principles will be rejected without discussion.

This ensures the project can remain secure, consistent, and responsibly governed.
